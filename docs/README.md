# 🚀 기능 목록

## 게임의 진행과 관련된 기능들

---

### a. 다리를 생성하는 기능
    다리 생성 시 U 또는 D를 저장하는 다리 길이 만큼의 리스트를 생성해야 한다.

### b. 다리 한 칸을 건너는 기능
    다리 위에서 한 칸 앞으로 간 후 이동할 수 있는 칸인지 여부를 리턴한다.

### c. 지금까지 이동한 경로를 초기화하는 기능
    이동할 수 없는 칸에 도달해 재시작을 선택하면 지금까지의 이동 경로가 초기화된다.
